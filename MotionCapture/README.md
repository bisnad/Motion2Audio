# Motion2Audio - MotionCapture

Th "Motion Capture" section provides tools for converting live motion capture data from proprietary to standard message protocols and for playing motion capture recordings.

The following tools are available:

- [MocapPlayer](https://github.com/bisnad/Motion2Audio/tree/main/MotionCapture/MocapPlayer)

  A python-based tool for playing recorded motion capture data and sending joint data via OSC

- [XSens2Osc](https://github.com/bisnad/Motion2Audio/tree/main/MotionCapture/XSens2Osc_FBX)

  A C++-based tool for converting the XSens native communication protocol to OSC
